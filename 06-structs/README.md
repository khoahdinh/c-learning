# 06 - Structs

Học cách tạo kiểu dữ liệu riêng bằng struct.

## Nội dung:

- **struct_intro.c** – giới thiệu struct.
- **typedef.c** – dùng `typedef` để đặt bí danh cho kiểu dữ liệu.
- **struct_array.c** – mảng các struct.

## Mục tiêu học:

- Tổ chức dữ liệu phức tạp trong chương trình.
- Hiểu cách truyền struct vào hàm.

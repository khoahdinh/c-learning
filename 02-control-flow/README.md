# 02 - Control Flow

Thư mục này bao gồm các cấu trúc điều khiển giúp bạn kiểm soát luồng chương trình.

## Nội dung:

- **if_else.c** – cấu trúc rẽ nhánh `if…else`.
- **switch_case.c** – xử lý nhiều trường hợp bằng `switch`.
- **for_loop.c** – vòng lặp `for`.
- **while_loop.c** – vòng lặp `while`.
- **do_while.c** – vòng lặp `do…while`.

## Mục tiêu học:

- Biết cách điều hướng luồng chương trình.
- Làm quen với các dạng vòng lặp khác nhau.
- Áp dụng vào các bài toán đơn giản.

# 07 - File Handling

Làm việc với file trong C: đọc – ghi – nối thêm nội dung.

## Nội dung:

- **read_file.c** – đọc file bằng `fscanf, fgets`.
- **write_file.c** – ghi file.
- **append_file.c** – ghi nối thêm vào cuối file.

## Mục tiêu học:

- Biết mở, đóng file với `fopen()`.
- Xử lý file text cơ bản.
- Hiểu các mode: `"r"`, `"w"`, `"a"`.

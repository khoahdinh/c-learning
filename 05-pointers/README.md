# 05 - Pointers

Đây là phần quan trọng nhất trong C – con trỏ.

## Nội dung:

- **pointers_basic.c** – khái niệm con trỏ.
- **pointer_arithmetic.c** – toán tử trên con trỏ.
- **pointer_and_arrays.c** – quan hệ giữa con trỏ và mảng.
- **pointers_to_functions.c** – con trỏ tới hàm.

## Mục tiêu học:

- Hiểu địa chỉ ô nhớ.
- Biết dùng con trỏ để tối ưu chương trình.
- Sẵn sàng cho struct, file handling và dynamic memory sau này.
